# Codex Desktop App Shortcuts

This is a quick reference for the Codex desktop app on macOS. These shortcuts are
for navigating the app UI, not for the `codex` CLI installed inside c0dev.

## Master Codex Skills

| Shortcut | Action |
| --- | --- |
| `Command+K` | Open command menu |
| `Command+P` | Search files |
| `Command+G` | Search chats |
| `Command+N` | Start a new chat |
| `Command+1` through `Command+9` | Jump to pinned chats |
| `Shift+Command+[` | Previous chat |
| `Shift+Command+]` | Next chat |
| `Command+J` | Toggle terminal |
| `Shift+Command+B` | Toggle browser panel |
| `Shift+Tab` | Toggle plan mode |
